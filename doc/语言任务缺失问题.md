## 词表说明

**当前实现为字符级词表**（Unicode code point → id），词表从语料文件自动构建，保存在 `<file>.vocab`（二进制，每项 4 字节 uint32_t code point）。
- **不需要下载外部词表**，适合字符级语言模型（中文约 3000-6000 字）
- 若需要 BPE/WordPiece 子词分词（GPT-2/Llama 风格），则需要外部词表文件和分词器（可对接 HuggingFace tokenizers）
